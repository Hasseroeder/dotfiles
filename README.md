# Dotfiles managed by Chezmoi

Initiate on a new system with:
```zsh
chezmoi init --apply git@github.com:Hasseroeder/dotfiles.git
```

Or clone with
```zsh
git clone https://github.com/Hasseroeder/dotfiles.git
```
